macOS 13+ · Apple Silicon & Intel · DMG install

First launch blocked? macOS blocks apps that aren’t notarized with Apple. After installing:

  1. Open Terminal and paste: curl -fsSL https://raw.githubusercontent.com/nishilfaldu/sediment/master/scripts/macos-install.sh | bash
  2. Or: System Settings → Privacy & Security → Open Anyway (appears after the first block)
  3. Or: right-click Sediment.app → Open (don’t double-click the first time)

A notarized build (opens normally) is coming once Apple Developer signing is wired up.
